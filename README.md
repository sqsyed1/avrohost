## What Is AvroHost?

AvroHost is run by Avromic LLC, a company founded in 2014 in Yerevan, Armenia. Their servers are located in Germany, which means decent connectivity for European audiences in particular.

The pitch is pretty simple: balance of price, speed, and quality. Whether that actually holds up depends a lot on what plan you're looking at.
